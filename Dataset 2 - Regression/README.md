# Final Term Project – Regression Modeling


## Deskripsi
Notebook ini bertujuan untuk memprediksi variabel target dari data transaksi menggunakan pendekatan **machine learning berbasis regresi**.  
Model utama yang digunakan adalah **Ridge Regression** dengan proses hyperparameter tuning.

## Alur Proses
1. Import library
2. Load dataset hasil preprocessing
3. Pemisahan data (train & test)
4. Training model Ridge Regression
5. Hyperparameter tuning
6. Evaluasi model
7. Analisis performa model

## Metode yang Digunakan
- Ridge Regression
- Train-test split
- Hyperparameter tuning
- Evaluasi menggunakan:
  - Mean Squared Error (MSE)
  - R² Score

## Library yang Digunakan
- pandas  
- numpy  
- scikit-learn  
- matplotlib  

## Output
- Model regresi terlatih
- Nilai performa model
- Grafik dan metrik evaluasi

Notebook ini menunjukkan bagaimana data transaksi dapat digunakan untuk **prediksi berbasis regresi**.
